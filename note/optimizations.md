## Optimization Complexity Based on Input Size

generally

| **Input Size (n)** | **Time Complexity**  |
|:------------------:|:-------------------:|
|      10..11        |     O(n!), O(n⁶)     |
|      15..18        |   O(2ⁿ * n²)         |
|      18..22        |   O(2ⁿ * n)          |
|        25          |   O(2ⁿ)              |
|       100          |   O(n⁴)              |
|       400          |   O(n³)              |
|      2000          |   O(n² * log n)      |
|      5000          |   O(n²)              |
|      10⁵           |   O(n log n)         |
|    5 * 10⁶         |   O(n)               |
|      10¹²          |   O(√n)              |
|      10¹⁸          |   O(log n)           |
